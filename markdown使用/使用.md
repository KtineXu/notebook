# markdown 使用

- 增强预览
  - 预览插件：markdown-preview
      - 支持预览实时渲染。(Ctrl + Shift + M)
      - 支持Latex公式。(Ctrl + Shift + X)
-  同步滚动
  - 插件：markdown-scroll-sync
    - 安装即用
- 图片粘贴
  - 插件 ： markdown-image-paste
      - 使用截图工具将图片复制到系统剪切板。
      - 在Markdown新起一行输入文件名。
      - Ctrl + V 会自动把图片保存到Markdown文件相同目录下(因此要求Markdown文件应该先保存)，并命名为刚输入的文件名，同时在刚才输入文件名行处生成img标签。
- pdf 导出
  - 插件：markdown-pdf
      - 使用：packages->markdown-pdf
- 汉化版
  - simplified-chinese-menu
  - 安装之后即可使用
